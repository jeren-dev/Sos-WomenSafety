# 🚨 SOS Women Safety App

An Android application built using Jetpack Compose that helps women stay safe by sending emergency alerts quickly during dangerous situations.

---

## 📱 Features

* 🔴 **SOS Button** – Instantly send emergency alerts
* 📍 **Location Sharing** – Sends real-time location to trusted contacts
* 📞 **Emergency Contacts** – Predefined contacts for quick access
* 🎤 **Voice Activation (Planned)** – Trigger SOS using voice command
* 🔒 **Secure & Fast** – Designed for quick response in critical situations

---

## 🛠️ Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **SQLite Database**
* **Android SDK**

---

## 🚀 How It Works

1. User presses the SOS button
2. App fetches current GPS location
3. Sends alert message + location to saved contacts
4. (Optional) Future: Voice trigger activation

---

## 📂 Project Structure

* `app/` → Main application code
* `sqlite/` → Local database handling
* `ui/` → Jetpack Compose UI components

---

## 🔐 Permissions Required

* Location Access
* SMS Permission
* Internet Access
